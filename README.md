## Overview
This project is a **website for Book Haven Bookstore**, designed as part of a web development coursework assignment. The website includes a **Homepage, Gallery, About Us/Contact Page, and a Community Page**. The project follows **HTML, CSS, and JavaScript** best practices while maintaining accessibility and usability.

## Features
- **Navigation Bar**: Available on all pages.
- **Homepage**: Displays the bookstore logo, name, bestsellers, and promotional content.
- **Gallery Page**: Features a **3x3 product grid** with "Add to Cart" buttons.
- **About Us/Contact Page**: Contains business hours and a **custom order/feedback form**.
- **Community Page**: Includes **customer testimonials** and bookstore community involvement.
- **Shopping Cart Feature**: Users can add items to a cart, view their cart, and clear/process orders.
- **Footer Section**: Social media links, newsletter signup, and navigation.

## Technologies Used
- **HTML5**: Structure and content.
- **CSS3**: Styling and layout.
- **JavaScript (JS)**: Dynamic features and interactive elements.
- **Figma**: Wireframing and prototyping.

## Setup Instructions
1. **Clone the Repository**
   ```sh
   git clone https://github.com/yourusername/book-haven.git
   ```
2. **Navigate to the Project Directory**
   ```sh
   cd book-haven
   ```
3. **Open index.html in a Browser**
   - You can double-click `index.html` to view the site in your preferred browser.
